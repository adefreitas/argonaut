# About

Rewrite of Archon in Go. Composes frames from a group of given images, and generates a video out of them

## External dependencies

### ffmpeg

- MacOS:
  Run `brew install ffmpeg`
- Ubuntu
  Run `sudo apt-get install ffmpeg`
- Windows
  Run `?`

## Run

```
go run .
```
